WPLMS Academy Migrate helps users to migrate all Academy courses to WPLMS courses

== Description ==
WPLMS Academy Migrate plugin is a free plugin developed to help users to migrate all of their Academy courses to WPLMS courses. The users do not have to worry a lot and perform various steps to migrate their courses. All this is done by a simple click. With the courses all of the quizzes and the units will be migrated successfully with just one click.

Tutorial On how to setup and get started : <a href="https://vibethemes.com/documentation/wplms/knowledge-base/wplms-academy-courses-migration-plugin/">link</a>

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WPLMS Academy Migrate'
3. Activate WPLMS Academy Migrate from your Plugins page. 

= From WordPress.org =

1. Download WPLMS Academy Migrate.
2. Upload the 'wplms_academy_migrate' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate WPLMS Academy Migrate from your Plugins page.


== Frequently Asked Questions ==

= Does it require Academy theme? =

Yes, The plugin needs the Academy theme to be activated to work.

= Do I need the WPLMS theme to begin? =

No, You need the academy theme to begin but after migrating you need the WPLMS theme to see all the changes. 

= Can I revert changes if I don't like it after migrating? =

Yes, after migration you can check all the courses and if you feel its not correct and again wants to move to Academy theme, then you can click on revert changes button to revert all the changes.
